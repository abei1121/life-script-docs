## 1. 什么是人生运势历？
Life Script（人生运势历）是一款将庞大紫微斗数实战体系与中国传统黄历深度融合的个人专属运势决策系统。系统主打钦天四化与中州派逻辑，拒绝大众化与娱乐化，旨在为数字游民、独立开发者及全球命理爱好者提供严密的决策导航。

不同于千篇一律的通用日历，本系统将流年、流月、流日星盘与传统择吉宜忌完美交织，生成千人千面的动态专属历法。

## 2. 核心架构与 Web3 理念
本系统采用前沿的技术栈与独特的商业模型：

* **完全无广告与免订阅:** 摒弃传统 SaaS 订阅制，全站 100% 无广告。
* **NFT 通行证赋能:** 90% 的基础排盘与历法功能完全免费。剩余 10% 的高阶深度推演功能，仅针对持有 XIAOJIUCAI CLUB PASS 的 NFT 玩家开放。
* **极客级隐私 (No-Account 架构):** 纯依靠本地设备 ID 进行用户识别，彻底免去繁琐的注册流程，最大程度保护命理隐私数据。
* **多语言全球化:** 原生支持繁体中文、简体中文、英文、日文、韩文、俄文、越南文等 8 种语言。

## 3. 独家功能：AI 专属命理师
系统内置深度优化的 AI 交互引擎。用户无需自行组织语言，系统会自动将复杂的黄历数据与个人专属排盘转化为大语言模型（LLMs）可理解的高级上下文。这使得每一次 AI 解析都具备极高的数理逻辑，从根本上消除了 AI 算命常有的“幻觉”与随机性。

## 4. 常见问题 (FAQ)

**Q: Life Script 适合用来解决什么问题？**
A: 适合在面临重大事业进退、地理位置变动（如数字游民旅居选择）、财富管理时，作为有底层数据支撑的运势参考工具。
